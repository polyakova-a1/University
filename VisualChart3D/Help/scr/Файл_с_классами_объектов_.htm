Файл с классами объектов

По желанию можно задать файл с классами объектов. Данный файл может быть представлен в трех видах.

Один-к-одному
 Данный файл представляет собой матрицу Nx1, где N - количество объектов. В данном файле в столбец записывается название класса для каждого объекта по порядку. Пример представлен на рисунке 5


Рисунок 5 - Пример файла с классами объектов типа "один-к-одному"

Число объектов класса
 Данный файл может быть представлен двумя видами:

  • матрица Mx2, где M - количество классов. В первый столбец записывается количество объектов, во второй название класса.
  • матрица Mx1, где M - количество классов. В первый столбец записывается количество объектов


 Разделение по столбцам либо пробел, либо табуляция. Пример представлен на рисунке 6.

 
Рисунок 6 - Пример файла с классами объектов типа "число объектов класса"

Начало класса

Данный файл может быть представлен двумя видами:

  • матрица Mx2, где M - количество классов. В первый столбец записывается порядковый номер объекта (начиная с нуля) с которого начинается класс, во второй название класса.
  • матрица Mx1, где M - количество классов. В первый столбец записывается порядковый номер объекта (начиная с нуля) с которого начинается класс


 Разделение по столбцам либо пробел, либо табуляция. Пример представлен на рисунке 7.

  
Рисунок 7 - Пример файла с классами объектов типа "начало класса"

Для задания файла с классами объектов требуется в окне "Настройки исходных данных" выбрать пункт "Файл с классами объектов", выбрать нужный тип,  нажать на кнопку "Обзор" и выбрать соответствующий файл. На рисунке 8 представлен пример


Рисунок 8 - Выбор файла с классами объектов
